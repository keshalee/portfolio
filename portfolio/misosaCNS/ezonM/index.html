    지자체/전통시장

    2. - 전통시장 : 입주상인의 대한 이존엠 반응형무료홈페이지제작 교육 및 공급

반응형 웹이란?

RESPONSIVE WEB(반응형 웹)은 화면의 크기에 맞춰 자동으로 최적화되는
웹페이지를 말합니다. 이런한 반응형웹으로 구성된 홈페이지는 태블릿, 그리고 스마트폰에 이르기까지 자동으로
최적화되어 사용자의 기기에 표시됩니다. 즉, 기기별로 별도의 웹페이지를 구현할 필요가 없습니다.
